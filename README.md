# Java TestNG Selenium 

### Environment Setup

Add Lambdatest Credentails details in Config.properties

    ```
    LambdaTest_UserName
    LambdaTest_AppKey
    ```
    
### Running Tests

```
To run single test
    $ mvn test -P single

To run parallel test
    $ mvn test -P parallel

```
### Test Result Setup

Please navigate to https://automation.lambdatest.com/ page for test result.
